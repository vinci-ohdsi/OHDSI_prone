metafor.news {metafor}R Documentation

Read News File of the Metafor Package

Description

Read news file of the metafor-package.

Usage

metafor.news()

Details

The function is just a wrapper for news(package="metafor") which parses and displays the ‘NEWS’ file of the package.

Author(s)

Wolfgang Viechtbauer wvb@metafor-project.org http://www.metafor-project.org/

References

Viechtbauer, W. (2010). Conducting meta-analyses in R with the metafor package. Journal of Statistical Software, 36(3), 1–48. https://www.jstatsoft.org/v036/i03.

Examples
