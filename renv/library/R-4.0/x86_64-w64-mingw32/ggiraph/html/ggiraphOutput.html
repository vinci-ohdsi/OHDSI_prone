ggiraphOutput {ggiraph}R Documentation

Create a ggiraph output element

Description

Render a ggiraph within an application page.

Usage

ggiraphOutput(outputId, width = "100%", height = "500px")

Arguments

outputId

output variable to read the ggiraph from.

width

widget width

height

widget height
